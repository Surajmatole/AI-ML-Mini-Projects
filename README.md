# 🤖 AI & Machine Learning Mini Projects

Welcome to my AI & Machine Learning project repository!

## 📚 Projects Included

### 1. Student Performance Predictor
- Built using Linear Regression
- Data visualization with Matplotlib
- Model evaluation using Mean Absolute Error
- Predicts exam scores based on study hours

### 2. Iris Flower Classifier
- Built using k-Nearest Neighbors (k-NN)
- Feature scaling with MinMaxScaler
- Classification Report
- Confusion Matrix
- Achieved 97.78% Accuracy

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## 📂 Files

- `AI_ML_Mini_Projects.ipynb` – Complete notebook
- `requirements.txt` – Required Python libraries

## 🚀 How to Run

```bash
pip install -r requirements.txt
```

Then open the notebook in Google Colab or Jupyter Notebook and run all cells.
